# Domain Architecture

Visualize protein domain annotations overlaid on genes. Domains are rendered as colored rectangles within gene arrows.

import { Callout } from 'nextra/components'

<Callout type="info">
  Domains can be colored by name, source (Pfam, TIGRFAM), or any custom attribute.
</Callout>

## Features Shown

- **Domain Layer**: Protein domains rendered inside gene arrows
- **Color by Name**: Each unique domain gets a distinct color
- **Larger Genes**: Increased `geneHeight` to better show domain architecture
- **Qualitative Palette**: Using 'Prism' palette for distinct domain colors

## Configuration

```tsx
initialState: {
  showDomainLayer: true,
  domainPalette: { type: 'qualitative', name: 'Prism', numColors: 10, enabled: true },
  domainColorBy: 'domainName',
  geneHeight: 80,
}
```

## Data Format

### Domains (TSV/Parquet)
```
gene_id    domain_id    start    end      source    evalue    coverage
gene_A2    PF00005      10.0     200.0    pfam      1e-50     0.85
gene_A2    PF00664      250.0    400.0    pfam      1e-30     0.75
```

### Domain Metadata (optional)
```
domain_id    name              description
PF00005      ABC_tran          ABC transporter
PF00664      ABC_membrane      ABC transporter transmembrane region
```
