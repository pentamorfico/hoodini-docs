# 5. Nucleotide Links

Show DNA-level synteny between genomic regions.

## What's New

- **Region-based links** connecting DNA segments
- **Wider ribbons** showing conserved blocks
- **Color by identity** - gradient from low to high similarity

## Protein vs Nucleotide Links

| Protein Links | Nucleotide Links |
|--------------|------------------|
| Gene-to-gene | Region-to-region |
| Based on protein sequence similarity | Based on DNA sequence similarity |
| Point-to-point connections | Block-to-block ribbons |
| From BLAST/MMseqs2 | From nucmer/minimap2 |

## Code

```tsx
<HoodiniViz
  newickStr={newickStr}
  gffFeatures={gffFeatures}
  hoods={hoods}
  proteinLinks={[]}
  nucleotideLinks={nucleotideLinks}  // Add DNA links
  domains={[]}
  nucleotideLinkConfig={{ 
    colorBy: 'identity',
    useAlpha: true, 
    minAlpha: 0.2, 
    maxAlpha: 0.7 
  }}
/>
```

## Link Configuration

| Property | Type | Options | Description |
|----------|------|---------|-------------|
| `colorBy` | string | `'identity'`, `'source'`, `'target'` | How to color links |
| `useAlpha` | boolean | - | Vary transparency by identity |
| `minAlpha` | number | 0-1 | Minimum transparency |
| `maxAlpha` | number | 0-1 | Maximum transparency |
