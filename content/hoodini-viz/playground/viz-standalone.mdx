# Standalone (No UI)

Pure visualization without sidebar or toolbar. Perfect for embedding in custom applications.

import { Callout } from 'nextra/components'

## Features

- **No Sidebar**: Full-width visualization
- **No Toolbar**: Clean, minimal appearance
- **Interactive**: Pan, zoom, and tooltips still work
- **Embeddable**: Perfect for iframes or custom apps

## Configuration

```tsx
{
  showSidebar: false,
  showToolbar: false,
}
```

## Use Cases

- **Embedded Views**: In blog posts or articles
- **Custom Applications**: When you build your own controls
- **Presentations**: Clean screenshots or exports
- **Kiosk Mode**: Display-only installations
