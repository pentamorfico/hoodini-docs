# With Phylogenetic Tree

HoodiniViz with a phylogenetic tree showing evolutionary relationships. Tree labels can be colored by taxonomy.

## Features Shown

- **Phylogenetic Tree**: Rendered from Newick string
- **Tree Alignment**: Tree leaves match hood order
- **Label Alignment**: Labels aligned to the right
- **Ultrametric Toggle**: Switch between phylogram and cladogram views

## Configuration

```tsx
dataPaths: {
  newick: '/path/to/tree.nwk',
  gffParquet: '/path/to/gff.parquet',
  hoodsParquet: '/path/to/hoods.parquet',
  treeMetadataParquet: '/path/to/tree_metadata.parquet',
},
initialState: {
  ultrametric: false,  // true = cladogram, false = phylogram
  alignLabels: true,   // align labels to right edge
}
```

## Data Format

### Newick Tree
```
((speciesA:0.1,speciesB:0.15):0.05,speciesC:0.2);
```

### Tree Metadata (optional, for coloring)
```
leaf_id      phylum            species
speciesA     Proteobacteria    Escherichia coli
speciesB     Proteobacteria    Salmonella enterica
speciesC     Firmicutes        Bacillus subtilis
```
