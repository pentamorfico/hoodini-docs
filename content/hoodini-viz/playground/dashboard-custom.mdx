# Custom Palettes

Customize the appearance with different color palettes for genes, domains, tree labels, and more.

## Available Palettes

### Qualitative (Categorical)
- `Set1`, `Set2`, `Set3` - ColorBrewer sets
- `Vivid`, `Bold`, `Prism` - High contrast
- `Tableau10` - Tableau default
- `Pastel1`, `Pastel2` - Soft colors

### Sequential
- `Viridis`, `Plasma`, `Inferno`, `Magma`
- `Blues`, `Greens`, `Reds`, `Purples`
- `YlOrRd`, `YlGnBu`, `BuGn`

## Color By Options

| Layer | colorBy Options |
|-------|-----------------|
| Genes | `cluster`, `product`, `source`, `annotation` |
| Domains | `domainName`, `source` |
| Tree | Any metadata column (`phylum`, `species`, etc.) |
| Regions | `region_type`, `source` |

## Full Example

```tsx
import { HoodiniDashboard } from 'hoodini-viz';
import 'hoodini-viz/dist/hoodini-viz.css';

<HoodiniDashboard
  dataPaths={{
    newick: '/data/tree.nwk',
    gffParquet: '/data/gff.parquet',
    hoodsParquet: '/data/hoods.parquet',
    domainsParquet: '/data/domains.parquet',
    proteinMetadataParquet: '/data/protein_metadata.parquet',
  }}
  initialState={{
    // Gene coloring
    genePalette: { type: 'qualitative', name: 'Vivid', numColors: 12, enabled: true },
    geneColorBy: 'cluster',
    
    // Domain coloring
    domainPalette: { type: 'qualitative', name: 'Set1', numColors: 9, enabled: true },
    domainColorBy: 'domainName',
    showDomainLayer: true,
  }}
/>
```
