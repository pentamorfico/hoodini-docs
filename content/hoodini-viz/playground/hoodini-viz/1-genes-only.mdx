# 1. Genes Only

The simplest possible visualization - just genes without tree, links, or domains.
This example shows a realistic bacterial genomic neighborhood with 8 genomes from 
Enterobacteriaceae and Pseudomonadaceae families.

## Code

```jsx
import { HoodiniViz } from 'hoodini-viz';

<HoodiniViz
  gffFeatures={gffFeatures}
  hoods={hoods}
  proteinLinks={[]}
  nucleotideLinks={[]}
  domainsByGene={{}}
  showScrollbar={true}
  showRuler={true}
  geneLabelBy="ID"
/>
```
