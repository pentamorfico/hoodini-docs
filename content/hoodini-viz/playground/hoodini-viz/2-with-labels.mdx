# 2. With Labels

Genes with protein metadata for richer tooltips and labels. The `proteinMetadata` 
object provides additional information like product names and cluster assignments 
for coloring.

## Code

```jsx
import { HoodiniViz } from 'hoodini-viz';

<HoodiniViz
  gffFeatures={gffFeatures}
  hoods={hoods}
  proteinLinks={[]}
  nucleotideLinks={[]}
  domainsByGene={{}}
  proteinMetadata={proteinMetadata}
  showScrollbar={true}
  showRuler={true}
  geneLabelBy="cluster"
  geneLabelSize={14}
/>
```
