## All Features

| Feature | Status | Description |
|---------|--------|-------------|
| Phylogenetic Tree | ✅ | Shows evolutionary relationships |
| Gene Neighborhoods | ✅ | Gene arrows with direction |
| Protein Metadata | ✅ | Labels and cluster info |
| Tree Metadata | ✅ | Taxonomy for tree coloring |
| Protein Links | ✅ | Gene-level synteny |
| Nucleotide Links | ✅ | DNA-level synteny |
| Domains | ✅ | Protein domain annotations |
| Ruler | ✅ | Scale bar |
| Scrollbar | ✅ | Navigation |

## Complete Code

```tsx
import { HoodiniViz } from 'hoodini-viz'

<HoodiniViz
  // Tree
  newickStr={newickStr}
  
  // Core data
  gffFeatures={gffFeatures}
  hoods={hoods}
  
  // Links
  proteinLinks={proteinLinks}
  nucleotideLinks={nucleotideLinks}
  
  // Annotations
  domains={domains}
  
  // Metadata
  proteinMetadata={proteinMetadata}
  treeMetadata={treeMetadata}
  
  // Visual settings
  geneHeight={70}
  showScrollbar={true}
  showRuler={true}
  
  // Gene coloring
  genePalette={{ type: 'qualitative', name: 'Bold', numColors: 10, enabled: true }}
  geneColorBy="cluster"
  
  // Domain coloring
  domainPalette={{ type: 'qualitative', name: 'Prism', numColors: 10, enabled: true }}
  domainColorBy="domainName"
  
  // Link styling
  proteinLinkConfig={{ 
    colorBy: 'source_gene', 
    useAlpha: true, 
    minAlpha: 0.1, 
    maxAlpha: 0.5 
  }}
/>
```

## Data Summary

| Data Type | Format | Required |
|-----------|--------|----------|
| `newickStr` | Newick string | No |
| `gffFeatures` | Array of GFF objects | **Yes** |
| `hoods` | Array of hood definitions | **Yes** |
| `proteinLinks` | Array of link objects | No |
| `nucleotideLinks` | Array of link objects | No |
| `domains` | Array of domain objects | No |
| `proteinMetadata` | Object keyed by gene ID | No |
| `treeMetadata` | Object keyed by leaf ID | No |

## When to Use HoodiniViz vs HoodiniDashboard

| Use Case | Component |
|----------|-----------|
| Pre-processed data, custom UI | **HoodiniViz** |
| Load from parquet files | HoodiniDashboard |
| Need sidebar controls | HoodiniDashboard |
| Embedded in existing app | **HoodiniViz** |
| Full standalone app | HoodiniDashboard |
